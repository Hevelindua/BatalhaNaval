# BatalhaNaval
Mini game de Batalha Naval feito em HTML, CSS e JavaScript durante a disciplina de Desenvolvimento Web.

# Batalha Naval - Projeto Web

Este projeto é uma versão simples do jogo Batalha Naval, desenvolvido utilizando HTML, CSS e JavaScript durante a disciplina de Desenvolvimento Web 1.
O objetivo do jogo é encontrar os barcos escondidos no tabuleiro antes que o número de tentativas acabe.

# 🎮 Demonstração do Jogo

O jogo apresenta um tabuleiro com 8 posições.
O jogador deve clicar nas posições para tentar encontrar os barcos escondidos.

Se acertar, um barco aparece, já se errar, aparece água. 
O jogador possui 4 tentativas.
